# Mindrailo Global Arcade — Launch Candidate 1

## Product shift
Mindrailo now leads with free browser games for a global audience. Kids and curriculum content remain available as a dedicated learning area.

## New
- Global Games Arcade at `/games/`
- Persistent Quick Play launcher
- Global navigation prioritising Play Games
- Five new playable games: Chess, Connect Four, Sudoku, Number Merge and Classic Snake
- Existing Tic-Tac-Toe, Snakes & Ladders and Memory Match surfaced in the global arcade
- Original category-card interface and related-game discovery
- Game SEO metadata and sitemap entries
- Homepage redesigned around games, quizzes and learning
- Public npm registry lock file for Cloudflare

## Quality checks
- Kids catalogue validator passed: 31 games
- ESLint passed
- TypeScript passed
- Production build passed
- 84 static pages generated

## Known launch limitations
- Chess is local two-player practice. Castling, en passant and formal checkmate enforcement are not included in this release.
- Global games save no user account data.
